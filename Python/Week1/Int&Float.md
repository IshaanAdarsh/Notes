- Type Funtion
```python
age = 3
print(type(age))    # < class 'int' >
age = 3.14
print(type(age))    # < class 'float' >
```
### Arithmetic Operators:
```python
# Addition:       3 + 2
# Subtraction:    3 - 2
# Multiplication: 3 * 2
# Division:       3 / 2
# Floor Division: 3 // 2
# Exponent:       3 ** 2
# Modulus:        3 % 2
```

### Comparisons:
```python
# Gives bool value for the given operations
# Equal:            3 == 2
# Not Equal:        3 != 2
# Greater Than:     3 > 2
# Less Than:        3 < 2
# Greater or Equal: 3 >= 2
# Less or Equal:    3 <= 2
```
### Round Function:
Gives rounded off answer till particular floating point
```python
print(round(3.75))    # 4
print(round(3.75,1))  # 3.8
```
### string to int conversion:
```python
num1 = '100'
num2 = '200'
print(num1+num2)    #100200

num1 = int(num1)
num2 = int(num2)
print(num1+num2)    #300
```
